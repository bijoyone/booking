# Full stack Hotel Booking App

##Description:

```
Search hotel 
Select single hotel
Show Hotel Information
Room image gallery
Search Room
Choose Room
Reserve Room
Price calculate
Admin Dashboard
Room options
Avilable Room```




##Technologies and libraries Used:

For frontend
- React
- Router
- Axios - (for fetch items)
- Fontawesome - (for icons)
- React-date-range - (for date range picker)
- CSS


For backend:

- MongoDB
- Node.js
- Express
- JWT
- Mongoose
- Nodemon - auto server start
- Bcryptjs - password encrypt
- Cookies
- React Context API

For Admin: 

- Axios
- Recharts
- SASS
- Cloudinary - image upload
